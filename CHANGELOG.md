## [0.1.0] - 2024-12-10

### Added
- 字符串反转功能（reverse_string） - 快速反转任意字符串
- 单词计数功能（count_words） - 统计文本中的单词数量
- 标题格式转换功能（to_title_case） - 将文本转换为标题大小写
- 空白字符移除功能（remove_whitespace） - 移除文本中的所有空白字符
- 回文检测功能（is_palindrome） - 检测字符串是否为回文，支持忽略大小写和空格
- 完整的单元测试套件
- 项目文档（README、CONTRIBUTING、LICENSE）
- Python 包配置（setup.py）

### Documentation
- 添加了详细的 README 文档，包含安装指南和使用示例
- 添加了贡献指南（CONTRIBUTING.md）
- 添加了 API 参考文档
